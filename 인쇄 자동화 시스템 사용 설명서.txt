========================================
    인쇄 자동화 시스템 사용 설명서
========================================

1. 프로그램 소개
----------------
이 프로그램은 인쇄 의뢰서에 썸네일과 QR 코드를 자동으로 삽입해주는 도구입니다.
파일 선택 후 F3 키만 누르면 자동으로 처리됩니다.


2. 설치 방법
------------
1) AutoHotkey 설치 (https://www.autohotkey.com)
2) 제공받은 폴더를 원하는 위치에 복사
3) PrintAutomation.ahk 파일 실행


3. 사용 방법
------------
[초기 설정 - 처음 한 번만]
1) Ctrl+F3 키를 눌러 설정 프로그램 실행
2) "샘플 PDF 열기" 버튼으로 의뢰서 샘플 선택
3) 마우스로 썸네일과 QR 위치 조정
4) "저장" 버튼 클릭

[일반 사용법]
1) Windows 탐색기에서 파일 3개 선택:
   - 의뢰서 PDF (파일명에 "의뢰서" 반드시 포함)
   - 인쇄물 PDF (표지 파일은 자동으로 오른쪽 절반만 사용)
   - QR 코드 이미지 (PNG 또는 JPG)

2) F3 키 누르기

3) 완료 알림음 확인
   - 높은음(띵~): 성공
   - 낮은음(뚜~): 실패


4. 단축키
---------
F3       : 선택한 파일 처리
Ctrl+F3  : 설정 프로그램 열기
F1       : 도움말 보기


5. 주의사항
-----------
- 의뢰서 PDF는 파일명에 "의뢰서"가 반드시 포함되어야 합니다
- 한 번에 정확히 3개 파일을 선택해야 합니다
- 처리된 파일은 원본을 덮어씁니다 (백업 권장)


6. 문제 해결
------------
Q: F3을 눌러도 반응이 없어요
A: AutoHotkey 스크립트(PrintAutomation.ahk)가 실행 중인지 확인하세요
   시스템 트레이에 H 아이콘이 있어야 합니다

Q: "선택된 파일이 없습니다" 메시지가 나와요
A: 탐색기에서 파일을 먼저 선택(클릭)한 후 F3을 누르세요

Q: 처리가 실패했다고 나와요
A: 파일이 다른 프로그램에서 열려있는지 확인하세요
   PDF 뷰어를 모두 닫고 다시 시도하세요


7. 고급 설정
------------
settings.json 파일을 직접 편집하여 세부 조정 가능:
- thumbnail: 썸네일 위치와 크기
- qr: QR 코드 위치와 크기
- processing: 처리 옵션
