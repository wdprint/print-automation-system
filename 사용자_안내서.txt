========================================
    인쇄 자동화 시스템 v2.0 사용자 안내서
========================================

📌 이 프로그램은 무엇인가요?
----------------------------
인쇄 의뢰서에 썸네일과 QR 코드를 자동으로 삽입해주는 
스마트한 자동화 도구입니다.

수작업으로 하던 반복 작업을 단축키 하나로 처리합니다!


🚀 시작하기 (매우 간단!)
------------------------

[준비물]
1. 의뢰서 PDF (파일명에 "의뢰서" 포함)
2. 인쇄물 PDF (썸네일용)
3. QR 코드 이미지 (JPG 또는 PNG)

[실행 방법]
1. start_enhanced.py 더블클릭
2. 필요한 것들이 자동 설치됩니다
3. 프로그램이 자동으로 시작됩니다!


📋 기본 사용법
--------------

방법 1) 드래그 앤 드롭 (초보자용)
  1. 프로그램 실행
  2. 파일 3개를 마우스로 끌어다 놓기
  3. 자동 처리 완료!

방법 2) 단축키 사용 (고급 사용자용)
  1. Windows 탐색기에서 파일 3개 선택
  2. F3 키 누르기
  3. 자동 처리 완료!


⌨️ 필수 단축키 (3개만 외우세요!)
---------------------------------

F3           : 선택한 파일 처리
Ctrl+Alt+F3  : 설정 창 열기
F1           : 도움말 보기


🎨 프리셋 사용하기
------------------

Alt+1 : 프리셋 1 (기본 설정)
Alt+2 : 프리셋 2 (사용자 설정)
Alt+3 : 프리셋 3 (사용자 설정)
Alt+4 : 프리셋 4 (사용자 설정)

각 프리셋마다 다른 위치에 이미지를 삽입할 수 있습니다!


📊 대기열 시스템
----------------

여러 세트를 한 번에 처리하려면?

1. 첫 번째 세트 선택 → F3
2. 두 번째 세트 선택 → F3
3. 세 번째 세트 선택 → F3
   ...계속 추가...

→ 알아서 순서대로 처리됩니다!

Shift+F3 : 대기열 상태 확인


⚙️ 고급 기능
-------------

[백지 감지]
- 빈 페이지를 자동으로 건너뜁니다
- 설정에서 민감도 조절 가능

[썸네일 옵션]
- 여러 페이지 선택 가능 (1-3, 1,3,5 등)
- 흑백 변환
- 대비/선명도 조정

[처리 규칙]
- 파일명에 "표지"가 있으면 → 자동으로 오른쪽 절반만 사용
- 파일명에 "급행"이 있으면 → 우선 처리


🔧 설정 조정하기
----------------

1. Ctrl+Alt+F3 키로 설정 창 열기
2. 원하는 탭 선택:
   - 위치 설정 : 이미지 삽입 위치 조정
   - 백지 감지 : 백지 감지 옵션
   - 프리셋 관리 : 프리셋 저장/불러오기
   - 썸네일 옵션 : 이미지 처리 설정
3. 설정 후 "저장" 클릭


❓ 문제 해결
------------

Q: F3을 눌러도 반응이 없어요
A: 파일을 먼저 선택했는지 확인하세요
   시스템 트레이에 H 아이콘이 있는지 확인하세요

Q: "의뢰서를 찾을 수 없습니다" 오류
A: PDF 파일명에 "의뢰서"가 포함되어야 합니다
   예) 인쇄_의뢰서_2024.pdf ✓
       인쇄_주문서_2024.pdf ✗

Q: 이미지가 엉뚱한 곳에 들어가요
A: Ctrl+Alt+F3 → 위치 설정 → 샘플 PDF로 테스트

Q: 처리가 너무 느려요
A: 고급 설정 → 성능 옵션 → 멀티스레딩 켜기

Q: 백지인데 처리되요
A: 고급 설정 → 백지 감지 → 활성화 체크

------------

💡 유용한 팁
------------

1. 프리셋 활용하기
   - 자주 쓰는 설정을 프리셋으로 저장
   - Alt+숫자로 빠르게 전환

2. 대량 작업시
   - 파일명 규칙 통일 (의뢰서_날짜.pdf)
   - 폴더별로 정리 후 작업

3. 시간 절약 팁
   - QR 코드는 한 번 만들어 재사용
   - 백지 감지로 불필요한 페이지 제거
   - 처리 규칙으로 반복 작업 자동화
